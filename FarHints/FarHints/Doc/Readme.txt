FarHints v 1.0 plugin for FAR
-----------------------------

Плагин показывает различные свойства файла во всплывающей подсказке.

Фичи:

- Подсказка появляется автоматически в позиции мыши или текущей позиции
  курсора при перемещении клавиатурой. Альтернативно можно настроить
  вызов подсказки вручную.

- Поддерживаются суб-плагины для показа свойств файлов различных типов.

- Для зарегистрированных системных типов в подсказке отображается иконка.

- Для программ в подсказке отображается Version Info и иконка программы.

- Для картинок показывается эскиз картинки и ее размеры. Декодируются
  картинки через GDI+, размер картинки в хинте настраивается, поддерживаются
  Thumbnail'ы

- Для каталогов подсчитывается размер содержимого и количество файлов.

- Поддерживаются Unicode-имена файлов


Установка плагина:

Для FAR 1.7:

Плагин использует механизм макросов, для автоматического показа хинта в 
основном потоке Фара. Перед началом использования необходимо зарегистрировать 
макрос для вызова. По умолчанию предлагается использовать макрос на кнопку F23, 
отсутствующую на большинстве клавиатур. Если это у Вас ни с чем не конфликтует, 
просто введите в реестр файл Setup.reg. Версия для Far 2 не требует регистрации
макроса для автоматического вызова.


Для FAR 1.7 и FAR 2:

Плагин поддерживает масштабирования эскизов при помощи колеса мыши. Для 
использования этой возможности зарегистрируйте макросы из файла SetupWheel.reg
