FarHints v 1.0 plugin for FAR
-----------------------------

Плагин показывает различные свойства файла во всплывающей подсказке.

Фичи:

- Подсказка появляется автоматически в позиции мыши или текущей позиции
  курсора при перемещении клавиатурой. Альтернативно можно настроить
  вызов подсказки вручную.

- Поддерживаются суб-плагины для показа свойств файлов различных типов.

- Для зарегистрированных системных типов в подcказке отображается иконка.

- Для программ в подсказке отображается Version Info и иконка программы.

- Для картинок показывается эскиз картинки и ее размеры. Декодируются
  картинки через GDI+, размер картинки в хинте настраивается, поддерживаются
  Thumbnail'ы

- Для каталогов подсчитывается размер содержимого и количество файлов.

- Поддерживаются Unicode-имена файлов


Установка

Плагин использует механизм макросов, для вызова хинта в основном потоке Фара.  
Поэтому, перед началом использования необходимо зарегистрировать хоткей 
плагина и макросы для его вызова. По умолчанию предлагается использовать
хоткей "!" и макросы AltShift1 и AltShift2, если это у Вас ни с чем не 
конфликтует, просто введите в реестр файл Setup.reg.

Также плагин имеет ряд настроек, которые можно изменить с помощью файла
Settings.reg
