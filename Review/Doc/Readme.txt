Review for FAR 3
----------------

Просмотр изображений и медиафайлов для FAR3

Ключевые особенности:

- Основные форматы изображений декодируются через GDI+ самим плагином
- Поддерживаются субплагины формата PVD, совместимые с PictureView v1 и v2
- Медиафайлы поддерживаются через субплагин VideoBass.pvd (используются 
  библиотеки BASS и BASSVIDEO - http://www.un4seen.com/)
- Автоповорот изображений, поворот вручную, сохранение повернутых изображений
- Использование эскизов для быстрого листания
- Режим QuickView без торможения и побочных эффектов
- Все управление настраивается через макросы
