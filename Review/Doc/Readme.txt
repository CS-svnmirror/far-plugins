Review for FAR 3
----------------

Просмотр изображений и медиафайлов для FAR3

Ключевые особенности:

- Основные форматы изображений декодируются через GDI+ самим плагином
- Поддерживаются субплагины формата PVD, совместимые с PictureView v1 и v2
- Медиафайлы поддерживаются через субплагин DXVideo.pvd (используется DirectShow)
- Автоповорот изображений, поворот вручную, сохранение повернутых изображений
- Использование эскизов для быстрого листания
- Режим QuickView без торможения и побочных эффектов
- Все управление настраивается через макросы


-------------------------------------------------------------------------------
История изменений:

Ver 1.1

- Для просмотра медиа-файлов больше не требуются библиотеки BASS, субплагин 
  переименован в DXVideo.pvd. Старый VideoBass.pvd лучше удалить.

- При просмотре медиа-файлов добавлено управление громкостью: через ползунок,
  колесиком мыши и кнопками Вверх/Вниз.

- Если плагин настроен для перехвата View (F3), то при нажатии AltF3 или
  CtrlShiftF3 плагин не работает, открывается стандартный viewer

- Версия для x64

- Исправлено несколько ошибок
