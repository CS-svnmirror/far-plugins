
Ver 1.0.1,  23.06.2008

 + Добавлена поддержка BASS плагинов. Подробнее в файле readme.txt,
   в разделе "Поддерживаемые форматы"

 + Диалог "Information" показывает список загруженных BASS плагинов и
   их версии, а также список поддерживаемых Audio форматов.

Ver 1.0.2,  24.06.2008

 + Добавлено чтение/сохранение playlist'а в текстовом файле. 
   Это делается с помощью команд /Save и /Load, подробнее - в readme.txt.

   Чтобы настроить Far таким образом, чтобы при выборе файла *.m3u
   сразу начиналось его воспроизведение, добавьте в ассоциации:

   Mask: *.m3u   Command: np:"/Load=!\!.!"

 + Добавлена иконка в systray, показывающая состояние проигрывателя,
   и дающая базовые возможности по управлению:

   - Одиночный leftclick - Pause/Play
   - Меню по rightclick - Pause/Play/Stop/Next/Prev/Quit
   - Колесико мыши - изменение громкости

   Убрать/показать иконку можно с помощью команды:

   np:/Systray

Ver 1.0.3, 26.06.2008

 * Если в playlist проигрывателя были добавлены файлы, которые он не
   мог воспроизводить, то он останавливался на первом таком файле.
   Теперь все "плохие" файлы будут пропускаться, а остальные - играться.

 + Если в командной строке в качестве маски файлов указать одну "*", 
   то будут добавлены только файлы с известными расширениями.

 + Если выполняются команды Open/Add из меню плагина, и в панели есть
   выделенные каталоги то они будут просканированы и в playlist будут
   добавлены все файлы с известными расширениями.

 + При добавлении большого количества файлов по команде Open/Add
   рисуется прогресс-индикатор.

 + При завершении работы проигрывателя автоматически сохраняется текущий
   playlist и проигрываемый трек. При запуске, они, соответственно,
   восстанавливаются.

   playlist сохраняется в каталоге %AppData%\Noisy

 + Добавлена индикация о текущем треке и режимах работы проигрывателя
   в виде ballon tooltip (только если включена иконка)

   Включить/выключить показ tooltip'ов можно с помощью команды

   np:/tooltip

 + Добавлена возможность регулировать ряд параметров работы проигрывателя,
   пока только через реестр. См. файл Settings.reg

 + Вызов Noisy добавлен в контекст диалогов.

Ver 1.0.4, 

 + Добавлена поддержка потокового Audio через интернет протоколы. Имя потока
   должно начинаться с префиксов "HTTP://" или "FTP://". Для shoutcast вещания
   поддерживаются теги, отображаемые в виде tooltip'ов. Для обычных файлов
   в диалоге информации отображается прогресс закачки.

 + Команды Open/Add files теперь работают в контексте файловой панели 
   плагина FarFTP. Они добавляют в playlist URL файлов, что позволяет 
   прослушивать файлы с FTP предварительно не скачивая их.

   Ограничения: 
     - FTP сервер должен разрешать анонимный логин. 
     - Так как проигрыватель открывает новое соединение, FTP сервер должен
       разрешать минимум 2 соединения с одного IP.

 + Добавлены режимы циклического и случайного воспроизведения. Их переключение
   осуществляется с помощью команд:

   np:/repeat
   np:/shuffle

 + В диалог информации добавлена кнопка "Config" по которой вызывается меню
   настроек проигрывателя.

 + Команда /Goto дополнительно поддерживает переход вперед/назад на заданное
   число треков.

 + Расширена поддержка playlist'ов в формате m3u. Поддерживаются теги #EXTINF

 * Исправлено несколько ошибок

Ver 1.0.5

 + Поддержка MP3 тегов V2 а также тегов ряда других форматов (WMA/APE/FLAC).
   MP3 теги поддерживаются нативно, остальные - через BASS addon "tags",
   (к сожалению, не очень дружный с русскими тэгами)

 + Фоновое извлечение тегов для файлов текущего playlist'а. Диалог Playlist
   по умолчанию показывает заголовки треков вместо имен файлов

 + Расширены возможности управления списком воспроизведения из диалога Playlist:

   - Теперь это не меню, а диалог, соотв. работают все макросы
   - Текущий проигрываемый трек отмечается символом ">"
   - Shift-Enter   - начать воспризведение трека, не закрывая диалог
   - Shift-Home    - перейти к воспроизводимому треку
   - Ctrl-1        - спрятать/показать номера треков
   - Ctrl-2        - показывать заголовки треков/имена файлов
   - Del           - удалить текщий трек из списка   
   - Ctrl-Up       - переместить текущий трек вверх
   - Ctrl-Down     - переместить текущий трек ввниз

 + Unicode-версия плагина NoisyFar для FAR 1.8. Сам проигрыватель Noisy.exe
   изначально unicode'ный, поэтому он одинаковый для обоих версий плагинов.

 + Немного help'а.

 * Исправлено несколько ошибок

Ver 1.0.6

  + Поддержка глобальных hotkey'ев и мультимедийных клавиатур. По умолчанию,
    при начале воспроизведения устанавливаются hotkey'и на доп. клавиши:

      Play/Pause, Stop, NextTrack, PrevTrack
        а также:
      Alt-VolumeUp, Alt-VolumeDown, Alt-Mute - управление громкостью

    Если Вы хотите изменить эти умолчания (например, на Вашей клавиатуре нет
    мультимедийных клавиш) - настройте под себя и введите в реестр файл
    Hotkeys.reg (менеджер hotkey'ев coming soon)

    Включить/выключить поддержку глобальных hotkey'ев можно с помощью команды

    np:/hotkeys

    или через меню "Config"

  + В диалоге Playlist работает фильтрация при наборе, поддерживаются маски (*, ?)


Ver 1.0.7

  + В systray меню добавлен пункт Playlist, позволяющий выбрать трек из
    списка. Тот же список появляется при правом клике на иконке при
    удержании Ctrl. Тот же список появляется по команде:

    np:/playlist

  + В systray меню добавлен пункт Open, по которому открывается файловый
    диалог выбора файлов/каталогов. То же  действие делается по команде:

    np:/open

    Все команды можно привязать к глобальным hotkey'ям.

  + В командной строке после префикса "np:" можно вызвать не только команды
    проигрывателя, но и команды плагина. В отличие от команд проигрывателя
    они начинаются с двух слэшей "//" (или "--")

    Список команд плагина:

    //Open       - Начать воспроизведение текущего файла (или выделенных файлов)
    //Add        - Добавить текущий файл (или выделенные файлы) в playlist
    //Playlist   - Открыть диалог Playlist
    //Info       - Открыть диалог Information
    //About      - Открыть диалог About

    Команды проигрывателя и плагина можно любым способом совмещать в одной
    командной строке. Например, чтобы настроить "модальный" режим проигрывания
    добавьте в ассоциации на музыкальные файлы:

    np:"!\!.!" //info /stop

  + Проигрыватель не завершает свою работу по тайм-ауту пока в Far'е открыты
    диалоги Information или Playlist (используются новые команды: lock/unlock).


Ver 1.0.8

  + Новая фича: история playlist'ов. Когда в проигрыватель загружается новый
    playlist, предыдущий автоматически сохраняется в каталоге %AppData%\Noisy.
    Размер истории по умолчанию равен 100 и может настраиваться через файл 
    Settings.reg.

    Для выбора playlist'а из истории в systray-меню добавлен пункт History. 

    Для очистки истории playlist'ов используется команда:

    np:/ClearHistory

  * При сохранении playlist'а в формате m3u записываются длины треков


Ver 1.0.9

  + Управление громкостью и позиционирование с помощью "жестов" мыши :)
    Левый клик на иконке и последующее перемещение мыши вверх/вниз меняет
    громкость, а влево/вправо - позицию в треке.

  + В playlist'ах проигрывателя и плагина отображается длина трека

  + Через диалог Open можно загружать playlist'ы

  * Улучшено извлечение тэгов из файлов, загружаемых через FTP/HTTP протоколы

  + Если в меню плагина нажать кнопку Tab, появляется диалог ввода, через
    который можно выполнить любую команду Noisy. Эту фичу можно использовать
    для задания макросов, не зависящих от раскладки меню плагина. Все макросы
    из файла macro.reg переписаны в новом стиле. Добавлены новые макросы:

    Ctrl+Alt+Right - Seek forward
    Ctrl+Alt+Left - Seek backward
   
  * Выбор некоторых пунктов меню (например, изменение громкости) не приводит 
    к его закрытию. Данное поведение настраивается через файлы *.mnu.

  * При выходе из субменю плагина по Esc мы возвращаемся в основное меню. 

  * Команда /Seek исполняется асинхронно, изменение сопровождается tooltip'ом.

  * Команда /Goto поддерживает переход на трек по имени (короткому или полному)

  + Команда для управление балансом (также доступно через субменю Volume):

    np:/Balance=

  + Команда для принудительного вызова подсказки:

    np:/ShowInfo

  * Изменен алгоритм разбора командной строки. Теперь можно брать в кавычки
    не все команду, а только ее аргументы. Т.е. равноправны след. варианты:

    "/Goto=Имя файла"
    /Goto="Имя файла"

  * Исправлено несколько ошибок
