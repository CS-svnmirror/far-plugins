Noisy Player v 1.0 for FAR
--------------------------

Noisy - Проигрыватель музыкальных файлов на основе BASS Audio Library.
Проигрыватель не имеет графического интерфейса и управляется через
командную строку или с помощью плагина для Far Manager.

NoisyFar - Plugin для Far Manager, для управления проигрывателем через
меню и диалоги Far или с помощью клавиатурных комбинаций, назначаемых
через макросы.

Благодаря разделению проигрывателя и управляющего plugin'а возможно
управление воспроизведением и playlist'ом из любой из запущенных копий 
Far'а. Кроме того, воспроизведение не прекращается если закрыть Far.

---

Подерживаемые форматы.

Для работы проигрывателя необходимо наличие библиотеки bass.dll версии 2.4, 
которая может находится в каталоге проигрывателя или любом другом, 
доступном через пути.

Данная библиотека изначально поддерживает следующие форматы:

  - Wave
  - MPEG layer 1/2/3
  - Ogg Vorbis
  - Audio Interchange Format

а так-же ряд музыкальных форматов: MOD, MTM, S3M, XM, IT.

Поддержка других форматов (WMA, APE, FLAC и т.п.) осуществляется с помощью
расширений, которые можно скачать с сайта разработчика:

  http://www.un4seen.com

По умолчанию проигрыватель ищет и пытается загрузить известные ему расширения
(basswma.dll, bassflac.dll, bass_ape.dll), которые могут находится там же
где и bass.dll.

Для загрузки любых других расширений необходимо указать их имена или каталог
размещения, прописав их в соответствующий ключ реестра, через файл Settings.reg

---

Формат командной строки:

 noisy [Ключи...] [Файлы...]

Допустимо указание нескольких имен файлов или масок файлов. По умолчанию
указанные файлы замещают текущий playlist и сразу начинают проигрываться.
Некоторые ключи командной строки могут менять это поведение.

Основные ключи:

/Add    Добавить файлы в playlist

/Play   Начать (продолжить) воспроизведение.

/Stop   Остановить воспроизведение

/Pause  Временно приостановить воспроизведения текущего трека

/Next   Перейти к следующему треку

/Prev   Перейти к предыдущему треку

/Quit   Завершить работу проигрывателя

Полный список ключей перечислен в файле NoisyCmd.txt

---

Управление проигрывателем из Far.

Плагин NoisyFar добавляет пункт "Noisy" в меню плагинов (F11) и префикс
командной строки "np:". В командной строке Far, после префикса "np:" 
возможно использование того же набора команд, что и в командной строке 
noisy.exe

Через меню плагина можно вызвать все те же команды еще одним способом.
Формат самого меню настраивается через текстовые файлы с расширением mnu.
Кроме вышеописанного набора команд в меню допустимы некоторые дополнительные
возможности:

Open
  Начинает воспроизведение текущего файла или множества выделенных файлов.
  Имеет смысл только в контексте файловой панели.

Add Files
  Аналогично, но файлы добавляются в конец playlist'а

Goto track
  Открывает меню с playlist'ом, для выбора проигрываемого файла

Information
  Открывает диалог с информацией о проигрываемом файле и базовыми 
  возможностями управления проигрывателем.


Для управления проигрывателем с помощью клавиатурных комбинаций удобно
назначить макросы, которые будут вызывать разные пункты из меню плагина.

В файле Noisy.lua предлагается некоторый набор таких макросов, который
показался мне удобным. Самостоятельно убедитесь, что предлагаемые комбинации 
не конфликтуют с теми, что Вы уже используете.
