
Ver 1.0.2

   автоматически при перемещению по тексту. Для сброса выделения добавлена
   команда Remove Highlight.

 + В диалоги поиска/замены добавлена опция "Reverse search".

 + В диалоге поиска добавлена кнопка "Count", для подсчета всех вхождений

 * Немного изменен дизайн диалогов поиска/замены

 * Исправлено несколько ошибок

Ver 1.0.6

 + Новый режим поиска - поиск всех вхождений (Grep). Фичи:
   - Высокая скорость
   - Дополнительная фильтрация в окне результатов
   - Автоматическая синхронизация редактора
   - Интеграция с FarHints (удобно просматривать длинные строки)

Ver 1.0.7

 + Поддерживаются регулярные выражения в строке замены: $0..$9

 * Исправлено несколько ошибок

Ver 1.0.8

 + Управление плагином через командную строку, передаваемую в CallPlugin.
   подробности - в хелпе.

 + Быстрый фильтр в окне Grep работает в XLAT режиме (опционально).

 + Опция Mark whole Tab, управляющая способом выделения Tab'ов

 + В диалогах Поиска/Замены работает клавиатурная команда CtrlP:
   вставить текущее слово (или выделение) из редактора

 + Изменен способ вызова через CallPlugin. Теперь его можно использовать
   для вызова любых команд плагина, в том числе - с диалогами.

 + Удалены команды Find word forward/backward - потому как элементарно
   реализуется через макросы.

 + Всем диалогам присвоены GUID
